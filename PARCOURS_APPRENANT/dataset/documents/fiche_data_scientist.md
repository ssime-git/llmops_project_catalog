# Fiche Métier — Data Scientist

## Définition

Le **Data Scientist** est responsable de l'analyse de données, de la modélisation statistique et du développement de modèles de machine learning pour résoudre des problèmes métier. Il combine compétences techniques (stats, ML, Python) et compétences métier (compréhension du problème, communication).

## Responsabilités clés

- Analyser les données pour en extraire des insights
- Développer et évaluer des modèles ML
- Communiquer les résultats aux parties prenantes
- Collaborer avec les équipes métier et techniques

## Compétences techniques

### Compétences obligatoires

| Compétence | Niveau attendu | Outils/Technos |
|------------|----------------|----------------|
| Statistiques | Avancé | tests d'hypothèses, distributions |
| Python | Avancé | pandas, numpy, scikit-learn |
| ML | Intermédiaire | sklearn, modèles classiques |
| Feature Engineering | Intermédiaire | sélection, création de features |
| Évaluation modèles | Intermédiaire | métriques, validation croisée |

### Compétences appréciées

- Deep Learning (PyTorch, TensorFlow)
- NLP (spaCy, Hugging Face)
- Visualisation (matplotlib, seaborn)
- MLflow (bases)

## Parcours type vers le métier

### Profil idéal de recrutement

- Formation en stats/maths/info (Bac+5)
- Expérience en analyse de données
- Connaissance des outils Python

### Parcours de formation (12-18 mois)

| Étape | Formation | Durée |
|-------|-----------|-------|
| 1 | Python pour la Data | 24h |
| 2 | Statistiques pour la Data Science | 20h |
| 3 | Introduction au Machine Learning | 32h |
| 4 | Feature Engineering et Sélection | 16h |

**Total** : ~92 heures

## Évolution possible

- **Senior Data Scientist** : Expertise technique approfondie
- **Data Scientist Lead** : Encadrement d'équipe
- **Machine Learning Engineer** : Passage en production
- **Data Analyst** : Retour vers l'analyse pure
