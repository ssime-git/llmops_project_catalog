# Fiche Métier — MLOps Engineer

## Définition

Le **MLOps Engineer** (ou ML Engineer en production) est responsable du déploiement et de la maintenance des modèles de machine learning en production. Il constitue le pont entre la Data Science et le système de production, en garantissant que les modèles fonctionnent de manière fiable et performante.

## Responsabilités clés

- Déployer et surveiller les modèles ML en production
- Mettre en place des pipelines de ML automatisés (CI/CD pour ML)
- Gérer le cycle de vie des modèles (model registry, versioning)
- Détecter et gérer le drift (dégradation des performances)

## Compétences techniques

### Compétences obligatoires

| Compétence | Niveau attendu | Outils/Technos |
|------------|----------------|----------------|
| Docker | Avancé | Dockerfile, Docker Compose |
| CI/CD | Intermédiaire | GitLab CI, GitHub Actions |
| Suivi d'expérimentation | Intermédiaire | MLflow, DVC |
| Monitoring | Intermédiaire | Prometheus, Grafana, Evidently |
| Python | Avancé | Scripts, FastAPI |

### Compétences appréciées

- Kubernetes pour l'orchestration
- Kubeflow ou BentoML pour le serving
- Feature store (Feast, Tecton)
- A/B testing pour modèles

## Parcours type vers le métier

### Profils qui basculent vers MLOps

- **ML Engineers** qui veulent passer en production
- **DevOps** qui veulent se spécialiser en ML
- **Data Engineers** avec expérience ML

### Parcours de formation (12-18 mois)

| Étape | Formation | Durée |
|-------|-----------|-------|
| 1 | Fondamentaux DevOps pour Data | 16h |
| 2 | MLOps Fondamentaux | 24h |
| 3 | MLOps Avancé — Production | 32h |

**Total** : ~72 heures

Prérequis : compétences en Python, SQL, et notions de ML.

## Évolution possible

- **ML Platform Engineer** : Conception de la plateforme ML interne
- **ML Reliability Engineer** : Focus sur la fiabilité des systèmes ML
- **ML Engineer Senior** : Direction technique d'équipe ML
