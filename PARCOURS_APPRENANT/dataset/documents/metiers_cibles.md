## Data Analyst

### Définition
Analyse les données pour en extraire des insights et produire des rapports. Utilise SQL, Excel, et outils de visualisation.

### Compétences obligatoires

- SQL avancé
- Outils de visualisation (Tableau, PowerBI)
- Statistiques de base
- Excel avancé

### Compétences souhaitables

- Python basics
- Communication des résultats

### Formations recommandées SkillPath

| Formation | Durée | Prérequis |
|-----------|-------|----------|
| Python pour la Data | 24h | Aucun |
| SQL Avancé et Optimisation | 16h | SQL basics |
| Visualisation et Storytelling Data | 16h | Aucun |

---

## Data Scientist

### Définition
Développe des modèles de machine learning pour résoudre des problèmes métier. Combine stats, ML et communication.

### Compétences obligatoires

- Python avancé (pandas, numpy)
- Statistiques avancées
- Machine Learning (sklearn)
- Feature engineering
- Évaluation de modèles

### Compétences souhaitables

- Deep Learning (PyTorch)
- NLP
- Visualisation avancée

### Formations recommandées SkillPath

| Formation | Durée | Prérequis |
|-----------|-------|----------|
| Python pour la Data | 24h | Aucun |
| Statistiques pour la Data Science | 20h | Aucun |
| Introduction au Machine Learning | 32h | Python, Stats |
| Feature Engineering et Sélection | 16h | ML concepts |

## Data Engineer

### Définition
Construit et maintient les pipelines de données. Gère le stockage, le traitement et la qualité des données.

### Compétences obligatoires

- Python avancé
- SQL avancé
- Spark / PySpark
- Airflow
- Docker

### Compétences souhaitables

- Cloud (AWS/GCP)
- Kafka (streaming)
- dbt

### Formations recommandées SkillPath

| Formation | Durée | Prérequis |
|-----------|-------|----------|
| Python pour la Data | 24h | Aucun |
| SQL Avancé et Optimisation | 16h | SQL basics |
| Data Engineering avec Apache Spark | 40h | Python, SQL |
| Airflow et Orchestration | 16h | Python |
| Fondamentaux DevOps pour Data | 16h | Git, Docker |

## ML Engineer / MLOps

### Définition
Déploie et maintient les modèles ML en production. Gère le cycle de vie, le monitoring et le drift.

### Compétences obligatoires

- Docker avancé
- CI/CD pour ML
- Suivi d'expérimentation (MLflow)
- Monitoring (Prometheus, Grafana)
- Python avancé

### Compétences souhaitables

- Kubernetes
- Kubeflow
- A/B testing

### Formations recommandées SkillPath

| Formation | Durée | Prérequis |
|-----------|-------|----------|
| Fondamentaux DevOps pour Data | 16h | Git, Docker |
| MLOps Fondamentaux | 24h | Docker, Python |

## LLM Engineer

### Définition
Déploie des systèmes RAG et LLM en production. Gère le prompt versioning, l'évaluation et le monitoring.

### Compétences obligatoires

- RAG (LangChain, LlamaIndex)
- Vector stores (Qdrant, pgvector)
- Prompt engineering avancé
- LLMOps (MLflow, prompt versioning)
- Évaluation (Ragas, DeepEval)

### Compétences souhaitables

- Fine-tuning (LoRA)
- Function calling
- Déploiement Ollama/vLLM

### Formations recommandées SkillPath

| Formation | Durée | Prérequis |
|-----------|-------|----------|
| LLM et RAG Fondamentaux | 24h | Python |
| LLMOps — Déploiement et Monitoring | 32h | LLM concepts, Docker |

## Conseils pour le parcours

### Pour démarrer en Data

1. Commencer par Python + SQL
2. Ajouter les statistiques
3. Se spécialiser (DS, DE, ou MLOps)

### Pour se spécialiser en ML/IA

1. Avoir des bases solides en Python et stats
2. Choisir son chemin : ML classique ou Deep Learning/LLM
3. Mettre en production rapidement (MLOps)

### Pour passer à LLM Engineer

1. Maîtriser les fondamentaux ML
2. Apprendre RAG et les vector stores
3. Travailler le prompt engineering
4. Mettre en place l'évaluation et le monitoring
